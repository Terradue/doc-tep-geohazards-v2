WPS providers managementÂ¶

The WPS providers management page contains a list of all existing WPS providers in DB, with the possibility for an Administrator to Create a new WPS provider, Update or Delete an existing one.
The Administrator can also Update a WPS provider (by clicking on ). This action will do a request to the WPS provider (GetCapabilities) to check wethere there are new WPS services available. If so, these services will be automatically added to the list of existing WPS services in DB.
